///NVM
Listar as versões disponíveis do Node.js:

nvm ls-remote
Como instalar
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
Instalar uma versão específica do Node.js:
nvm install versao_desejada
Usar uma versão específica do Node.js:
nvm use versao_desejada
nvm --version
